golang中的defer笔记

关键点

  • defer是在return之前执行的
  • return 语句并不是一条原子指令，return被拆成三条语句
  1. 返回值 = xxx
  2. 调用defer函数
  3. 空的return

例子

🤔 tips: 可以思考一下，然后运行看看 👀

func f() (result int) {
    defer func() {
        result++
    }()
    return 0
}

func f() (r int) {
     t := 5
     defer func() {
       t = t + 5
     }()
     return t
}

func f() (r int) {
    defer func(r int) {
          r = r + 5
    }(r)
    return 1
}

defer与闭包

如下case会出现空指针异常。

package main

import "fmt"

type s struct {
 i int
}

func bar(in *s) {
 fmt.Println("i is %d", in.i)
}

func foo() (ret *s) {
 bar(ret)
 ret = &s{
  i: 1024,
 }
 return
}

func main() {
 foo()
}

将foo函数改为如下，可以解决空指针：

func foo() (ret *s) {
 defer func() {
  bar(ret)
 }()
 ret = &s{
  i: 1024,
 }
 return
}
